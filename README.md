## Launching of project

  ##### Clone this repository
  > $ git clone https://github.com/Cluser/notifeed.git

  ##### Go into the repository
  > $ cd notifeed

  ##### Install dependencies
  > $ npm install

  ##### Start's development server
  > $ npm start

  ##### Navigate to http://localhost:4200/
